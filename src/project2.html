Crypto Trader

Manage your crypto investments in a fun and interactive way with our Crypto Portfolio Manager. Buy, sell, and track the performance of your favorite cryptocurrencies in real-time, leveraging the CoinCap API for live data. Dive into a user-friendly interface designed for both novices and seasoned investors.

Key Features

Live Price Updates: Real-time cryptocurrency data from the CoinCap API.

Interactive Charts: Visualize price trends with dynamic HTML canvas charts.

User Portfolio Management: Track your buying, selling, and overall portfolio performance.

Local Storage: Persistent user data for a seamless experience across sessions.
